
# URL SHORTEN APP 


## API Reference

#### Get all items

```http
  GET /api/shortUrl/${id}
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `id` | `string` | **Required** |


#### Get item

```http
  GET /api/allShortUrl
```



| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `none` | `none` | **none** |


```http
  POST /api/createShortUrl
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `fulUrl` | `string` | **Required** body |

```http
  DELETE /api/deleteShortUrl/${id}
```

| Parameter | Type     | Description                       |
